Row Class

This is a representation of a single row (record) in a table.

Column fields

The table shows the declared methods for the row column representations:

MethodDescription
ColumnType get()get the cell value
set(ColumnType value)set the cell value

Examples:

String name = people.first().name.get();
people.at(3).age.set(33);
