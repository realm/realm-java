View Class

The findAll method in the Query class returns a View object. You can interact with the View just as a regular table, and all changes to it will propagate to the original table.

Examples:

String[] names = people.name.getAll();
Person firstJohn = people.name.findFirst("John");
PeopleView hiredYoung = people.hired.equals(true).age.between(20, 30).findAll();
